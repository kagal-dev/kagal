# Kagal

> *Kagal* (𒅅𒃲) — Sumerian for "Great Gate"

A framework for managing agent fleets over Cloudflare's
edge.

## Packages

| Package | Description |
|---------|-------------|
| [`@kagal/worker`](packages/@kagal-worker) | Durable Object library |
| [`@kagal/server`](packages/@kagal-server) | Server library for frontends |
| [`@kagal/agent`](packages/@kagal-agent) | Agent CLI and library |
